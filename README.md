# push swap

> Projet réalisé à l'École 42 Paris

## 📖 Description
Algorithme de tri d'une pile d'entiers en utilisant deux piles et un ensemble limité d'opérations, avec un minimum de mouvements.

## 🛠 Compétences développées
- Algorithmique et optimisation
- Structures de données (piles)
- Complexité et performance
- Programmation en C

## 🚀 Compilation
```bash
make
./push_swap [liste de nombres]
```
